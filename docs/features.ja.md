# 機能詳細

## ノードの追加

| 種別 | 説明 |
|------|------|
| **人物** | 氏名・役割（容疑者/被害者/関係者/目撃者）・写真・説明を入力して追加 |
| **証拠** | 証拠名・種別（物証/写真/映像/文書/凶器/DNA/指紋）・画像・説明を追加 |
| **メモ** | タイトルとコメントテキストを追加 |

## キャンバス操作

| 操作 | 方法 |
|------|------|
| ノードの移動 | ドラッグ&ドロップ |
| ノード同士を接続 | ノード端の丸印をドラッグして別ノードの丸印へ（方向は矢印で表現） |
| 線の接続先を付け替え | 既存の線の端をドラッグして別ノードの丸印へ移動（ラベル・矢印設定は維持） |
| 関係性ラベルの設定 | 接続線をダブルクリック → プリセット（共犯/知人/目撃 等）から選択、またはフリー入力 |
| 矢印の向き変更 | 接続線をダブルクリック → 「→ 片方向」「← 逆方向」「↔ 双方向」「― なし」から選択 |
| ノードの編集 | ノードをダブルクリック → 名前・役割・写真・説明を変更 |
| 複数選択 | Shift + クリック |
| 選択削除 | 選択後に Delete キー、または右上の「選択削除」ボタン |
| ズーム | マウスホイール |
| パン（スクロール） | キャンバスの空白をドラッグ |
| 全体表示に戻す | 画面左下のコントロールにある「枠に収める」ボタン |
| ミニマップ | 右下に全体の縮小表示（人物=青、証拠=黄、メモ=灰で色分け） |

## 人物カードの色分け

| 役割 | 色 |
|------|------|
| 容疑者 | 赤 |
| 被害者 | 青 |
| 関係者 | 黄 |
| 目撃者 | 緑 |

## 画像の自動圧縮

写真をボードに追加する際、自動的に以下の処理が行われます。

| 項目 | 内容 |
|------|------|
| リサイズ | 最大 800x800 ピクセルに縮小（カード表示は180px幅のため十分な画質） |
| 圧縮形式 | JPEG 品質80% で再エンコード |
| 圧縮後サイズ | 1枚あたり約 50〜100 KB |

これにより、スマホやデジカメの高画質写真（3〜10MB）をそのまま貼り付けても、ボードの動作が重くならず、保存ファイルのサイズも小さく抑えられます。写真100枚程度でも保存ファイルは約10MB以下に収まります。

## データ管理

| 操作 | 説明 |
|------|------|
| 保存 | ボードの状態をJSONファイルとしてダウンロード（写真データも含まれる） |
| 読込 | 保存したJSONファイルからボードを復元 |
| クリア | ボードの全内容を削除（確認ダイアログあり） |

## 多言語対応

- 日本語 / 英語の切り替えに対応（サイドバーの JA / EN ボタン）
- ブラウザの言語設定を自動検出して初期表示言語を決定
- 選択した言語は localStorage に保存され、次回アクセス時も維持
